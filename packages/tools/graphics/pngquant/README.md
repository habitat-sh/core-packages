# Habitat package: pngquant

Lossy PNG compressor.

## Maintainers

The Habitat Maintainers (humans@habitat.sh).

## Type of package

Library package. Include this plan as a runtime or build dependency for plans that require `pngquant`.

## Usage

```
pkg_deps=(
  core/pngquant
)
```

or

```
pkg_build_deps=(
  core/pngquant
)
```

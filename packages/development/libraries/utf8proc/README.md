# utf8proc

A clean C library for processing UTF-8 Unicode data.

## Maintainers

* The Habitat Maintainers <humans@habitat.sh>

## Type of Package

Binary package

## Usage

This is a binary package of a library. Include this in your `pkg_deps` or `pkg_build_deps` as necessary.

```
pkg_build_deps=(
  core/utf8proc
)
```
